# @humanix/pdf-templates — Gabarits PDF versionnés

Génération serveur **déterministe** (Playwright/Chromium headless ou pdf-lib). Gabarits
**versionnés** : feuille d'**émargement** conforme (mentions obligatoires + lien/QR de
vérification), **certificat de réalisation** (gabarit DGEFP), **décompte de facturation**.
Tout artefact est horodaté, intègre et vérifiable.

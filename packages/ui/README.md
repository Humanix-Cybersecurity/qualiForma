# @humanix/ui — Design system accessible

Composants basés sur des primitives **headless accessibles** (React Aria privilégié, Radix en
repli) + Tailwind. Aucun widget interactif maison non accessible. Cible RGAA 4.1 / WCAG 2.1 AA,
focus visible, préférences (contraste, taille, `prefers-reduced-motion`). Voir
[ADR 0004](../../docs/adr/0004-accessibilite-rgaa-gate-ci.md).
